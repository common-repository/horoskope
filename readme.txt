Mit diesem PlugIn bekommen Sie jeden Tag neue Liebes-, Partner- und Tageshoroskope.

== Description ==

Dieses PlugIn ist ein Teil der AfiliTxt- PlugIn- Serie.

Dieses PlugIn holt jeden Tag neue Horoskope von einem AfiliTxt- Server. Diese Horoskope k&ouml;nnen mit einem Widget oder mit einem Platzhalter
in Artikeln und Seiten angezeigt werden.

== Installation ==

So installieren Sie das Horoskop PlugIn.

1. Installieren Sie das PlugIn in den Ordner `/wp-content/plugins/`
2. Aktivieren Sie das PlugIn im 'Plugins' Men&uuml; in WordPress
3. Klicken Sie im AdminMen&uuml; auf den Eintrag 'Horoskop' und holen Sie sich einen kostenlosen Account bei AfiliTxt &uuml;ber den Button 'Account beantragen'
4. Klicken Sie auf die Button Liebeshoroskope, Partnerhoroskope und Tageshoroskope, um die ersten Horoskope vom AfiliTxt- Server herunter zu laden.
5. Erstellen Sie ein oder mehrere Widget oder platzieren Sie einen der angegebenen Platzhalter in einen Artikel oder eine Seite

== Frequently Asked Questions ==

= Was kostet die Benutzung des PlugIn? =

Die Benutzung dieses PlugIn ist 100% kostenlos. AfiliTxt beh&auml;lt sich aber das Recht vor, in den Horoskopen Werbelinks zu platzieren.

= Wie bekomme ich neue Horoskope f&uuml;r den n&auml;chsten Tag? =

Das PlugIn holt sich automatisch alle 24 Stunden neue Horoskope von AfiliTxt- Server.

= Wo kann ich mir ansehen, wie das PlugIn in einem Wordpress Blog aussieht? =

Auf der Webseite <a href="http://www.horoskop-online.net" target="_blank" alt="horoskope">http://www.horoskop-online.net</a> wird gezeigt, wie die Horoskope auf einer Seite verwendet werden k&ouml;nnen.

= Ich habe eine Frage! Wo werden mir diese Fragen beantwortet? =

Alle Fragen zum PlugIn beantworten wir gerne auf unserer <a href="http://www.facebook.com/HoroskopPlugin" target="_blank" alt="facebook horoskope">Facebook Seite</a>.
